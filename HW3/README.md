# 구현
1. 입력 정의  
`A 위치(lat/lon), B 위치(lat/lon), 비교 이벤트(A.sunrise/B.sunset 등), 기간(start/end), 허용 오차(±10분), 시간대(GMT/UTC)`를 고정한다.

2. 데이터 수집  
`utils/open_meteo_sun.py`로 A/B 각각 `daily.sunrise`, `daily.sunset`를 조회하고 날짜별로 정렬한다.

3. 시간 정규화  
API 시각(`YYYY-MM-DDTHH:MM`)을 UTC 기준 `datetime`으로 변환해 절대 시점으로 비교한다.

4. 매칭 판정  
`diff_min = |tA - tB|`(분) 계산 후 `diff_min <= 10`이면 매칭으로 저장한다.

5. 정렬/출력  
여러 매칭이 있을 수 있으므로 `최신 연도 우선 -> 같은 연도면 diff 오름차순 -> diff 같으면 더 최근 시점 우선`으로 정렬해 CSV/콘솔로 출력한다.

  --lat 45.5017 --lon -73.5673 \
  --start 2026-03-01 --end 2026-03-05 \
  --timezone GMT \
  --output HW3/montreal_sun_times.csv
